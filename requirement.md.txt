# Education Performance and Cost Tracking Application

## Product Requirements Document (PRD) – Phase 1

# 1. Project Overview

Develop a cross-platform Education Performance and Cost Tracking application for parents to monitor and analyze the academic performance, educational expenses, and teacher effectiveness for multiple children.

The application shall be developed entirely in Python and must support:

* Android OS (Primary target for Phase 1)
* Windows OS Desktop Application
* Shared codebase for both platforms
* Local offline operation without internet dependency
* Future migration to cloud/server architecture

The first deliverable is an Android APK application with support for Windows desktop execution.

---

# 2. Business Objectives

The application should help parents to:

1. Digitize and maintain historical academic records.
2. Automatically extract marks from scanned progress reports.
3. Analyze academic progress over years.
4. Track educational expenses for each child.
5. Predict future educational costs.
6. Measure teacher effectiveness based on student performance.
7. Maintain all educational information in a single application.
8. Operate completely offline while maintaining privacy.

# 7. Progress Report Scanning Module

## Input Methods

* Camera capture
* Gallery image
* PDF upload

Supported formats:

* JPG
* PNG
* PDF

---

## OCR Processing Workflow

Step 1:
User selects child.

Step 2:
User selects:

* Academic Year
* Class
* Exam Type

Step 3:
Capture image or upload report.

Step 4:
OCR extracts:

* Subject names
* Marks obtained
* Maximum marks
* Grade
* Percentage
* Rank
* Remarks

Step 5:
Application displays extracted data.

Step 6:
Parent verifies and corrects OCR mistakes.

Step 7:
Data is stored permanently.

---

## OCR Accuracy Requirements

Target accuracy:

* Printed reports: 95%+
* Digital PDF reports: 99%+

Manual correction option must always be available.

# 22. Success Criteria

The project will be considered successful when:

1. Parent can enroll children.
2. Progress report scanning achieves at least 95% OCR accuracy.
3. Fee receipt scanning works reliably.
4. Historical data can be exported and imported.
5. Academic trends can be visualized.
6. Educational expenses can be forecasted.
7. Teacher effectiveness can be analyzed.
8. The application runs fully offline on Android and Windows.
